# Multilingual News Topic Classifier

An NLP classification template using TF-IDF + Linear SVM with a Streamlit demo app. Includes training, evaluation, and prediction utilities.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python scripts/train_model.py
```

## Run the Demo

```bash
streamlit run app.py
```
