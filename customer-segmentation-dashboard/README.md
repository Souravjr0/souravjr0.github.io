# Customer Segmentation & Insights Dashboard

A clustering workflow with K-Means and an interactive Streamlit dashboard to explore customer segments.

## Quick Start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python scripts/train_model.py
```

## Run the Dashboard

```bash
streamlit run app.py
```
