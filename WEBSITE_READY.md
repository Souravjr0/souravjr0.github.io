## ⚠️ Important Notes

### What GitHub Pages Can Do ✅
- Host static HTML/CSS/JS websites
- Serve images, videos, PDFs
- Use custom domains
- Enable HTTPS automatically
- Deploy instantly from GitHub
- Run 24/7 with 99.99% uptime

### What GitHub Pages Can't Do ❌
- Run Python/Node.js backends
- Connect to databases directly
- Process form submissions (use external service)
- Run scheduled jobs (use GitHub Actions instead)

**For interactive features** (forms, database, etc.):
- Add external services (Formspree, Basin, etc.)
- Use GitHub Actions for automation
- Embed interactive tools from other services
